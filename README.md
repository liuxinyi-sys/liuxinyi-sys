## Featured Projects

### pico - Local Coding Agent

一个面向代码仓库的轻量本地 coding agent，支持读取工作区、执行受约束工具、保存会话状态，并接入多类大模型后端。

- 支持 `OpenAI-compatible`、`Anthropic-compatible`、`DeepSeek`、`Ollama`
- 具备会话存储、运行记录、工具执行、上下文管理等模块
- 配套测试覆盖 agent loop、工具执行、安全约束、上下文管理等关键路径

### SQLOmni / SQL Audit Platform

围绕 SQL 审核与数据库对象分析的工程实践项目，包含前端管理界面、后端服务、审核规则与任务处理流程。

- 前端：`Vue` / `TypeScript` / `Element Plus`
- 后端：`Python` / `Flask` / `Redis`
- 场景：SQL 审核、任务管理、规则配置、结果分析
